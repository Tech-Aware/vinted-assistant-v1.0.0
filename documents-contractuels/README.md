# Pack initial — Documents contractuels de collaboration

Ce dossier contient une **base prête à versionner** pour démarrer un dépôt dédié aux documents contractuels.

## Contenu

1. `01-charte-collaboration.md` — règles de fonctionnement entre partenaires.
2. `02-nda-confidentialite.md` — accord de confidentialité bilatéral.
3. `03-contrat-prestation-tech.md` — contrat type de prestation technique (développement/IA).
4. `04-avenant-modele.md` — trame d’avenant pour modifier un contrat existant.

## Mode d’emploi rapide

1. Remplacer tous les champs entre crochets (`[ ... ]`) par vos informations.
2. Valider les clauses juridiques avec un professionnel du droit (France/UE ou juridiction applicable).
3. Ajouter une version PDF signable si nécessaire.
4. Tagger une version Git (`v0.1-docs`, `v1.0-signable`, etc.) pour le suivi.

## Convention de version

- `v0.x` : brouillon
- `v1.0` : version validée juridiquement
- `v1.x` : modifications mineures (dates, contacts, annexes)
- `v2.x` : changements contractuels substantiels

## Important

Ces modèles sont une base opérationnelle et ne remplacent pas un conseil juridique personnalisé.
